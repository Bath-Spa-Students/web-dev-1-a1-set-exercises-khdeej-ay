HTML & CSS Reference Guide

A quick reference guide for beginner web developers. This guide contains the basics of two core web development languages: HTML and CSS.

HTML - HyperText Markup Language

HTML (HyperText Markup Language) is the most basic building block of the Web. It defines the meaning and structure of web content.

HTML Elements

HTML elements tell the browser how to display the content. Most HTML elements contain three main parts: opening tag, content and closing tag.

HTML tags that require both an opening and closing tag are called container tags. 

<p>This is a paragraph.</body>

HTML tags that do not need a closing tag are called empty tags 

<br>

HTML Boilerplate

A boilerplate in HTML is a template you will add at the start of your project. You should add this boilerplate to all of your HTML pages.

The Emmet Abbreviation shortcut for the boilerplate code is !.

The following is what basic HTML Boilerplate looks like:

Syntax

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document Title</title>
</head>
<body>
    Text Here
</body>
</html>

Basic HTML Tags

01 | Comments

HTML comments are not displayed in the browser, but they can help document your HTML source code.

<!--This is a comment-->

02 | Headings

HTML headings are titles or subtitles that you want to display on a webpage. HTML headings are defined with the <h1> to <h6> tags. <h1> defines the most important heading. <h6> defines the least important heading.

Code
<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>

03 | Paragraph

The <p> element allows you to create text paragraphs in your website. A paragraph always starts on a new line, and is usually a block of text.

Code
<p>This is a paragraph.</p>
<p>This is another paragraph.</p>

04 | Line Break

The <br> tag is an empty or void tag. It forces the next element to be displayed on a new line.

Code
<p>This is a paragraph.</p>
<br>
<p>This is another paragraph.</p>

05 | Horizontal Rule

The <hr> HTML element represents a section break between elements: for example, to separate topic changes. It is most often displayed as a horizontal rule that is used to separate content (or define a change) in an HTML page.

Code
<p>HTML is the standard markup language for creating Web pages.</p>
<hr>
<p>CSS is a language that describes how HTML elements are to be displayed on screen, paper, or in other media.</p>
<hr>
<p>JavaScript is the programming language of HTML and the Web.</p>

06 | Image

The <img> HTML element embeds an image into the document. The <img> tag has two required attributes:
  - src: Specifies the path to the image you want to embed.
  - alt: Specifies an alternate text for the image, if the image for some reason cannot be displayed

Code
<img src="media/coding.jpeg" alt="A coding illustration">

07 | Anchor Link

The <a> HTML element (or anchor element), with its href attribute, creates a hyperlink to web pages, files, email addresses, locations in the same page, or anything else a URL can address.

Code
<a href="https://example.com/">Website</a>

08 | Div container

The <div> tag defines a division or a section in an HTML document. The <div>is used as a container for HTML elements - which is then styled with CSS or manipulated with JavaScript. The <div> is easily styled by using the class or id attribute.

Code
<div class="warning">
    <p>Beware of the leopard!</p>
</div>

09 | Span Container

The <span> tag is an inline container used to mark up a part of a text, or a part of a document. The <span> tag is easily styled by CSS or manipulated with JavaScript using the class or id attribute.
The <span> tag is much like the <div> element, but <div> is a block-level element and <span> is an inline element.

Code
<p>Some
    <span>Important</span>
Text.</p>
